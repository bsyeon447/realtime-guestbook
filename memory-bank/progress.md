# 개발 진행 상황

*   [ ] **1단계: 프로젝트 초기 설정**
    *   [ ] server 폴더 생성 및 Node.js 프로젝트 초기화 (
pm init)
    *   [ ] client 폴더 생성 및 React 프로젝트 초기화 (create-react-app)

*   [ ] **2단계: 백엔드 API 및 소켓 서버 구축**
    *   [ ] Express 서버 기본 설정
    *   [ ] Socket.io 연동 및 기본 연결 테스트
    *   [ ] 방명록/댓글 데이터 모델 설계 (JSON 구조)
    *   [ ] 방명록 저장을 위한 파일 업로드(multer) 및 API 엔드포인트 구현
    *   [ ] 'new-post', 'new-comment' 소켓 이벤트 핸들러 구현

*   [ ] **3단계: 프론트엔드 UI/UX 개발**
    *   [ ] 메인 페이지: 방명록 작성 폼 컴포넌트 구현
    *   [ ] 그림판 기능 (Canvas) 컴포넌트 구현
    *   [ ] 방명록 목록 페이지: 포스트잇 그리드 UI 구현
    *   [ ] 상세 보기 모달 컴포넌트 및 댓글 UI 구현

*   [ ] **4단계: 프론트엔드-백엔드 연동**
    *   [ ] socket.io-client 설정 및 서버 연결
    *   [ ] 방명록 작성 (API 요청) 및 실시간 업데이트 로직 구현
    *   [ ] 댓글 작성 (소켓 이벤트) 및 실시간 업데이트 로직 구현
